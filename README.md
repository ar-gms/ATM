## ATM

Dies ist eine einfache Java-Anwendung, die eine Benutzeroberfläche für eine Bankkonto-Verwaltung bereitstellt. Die Anwendung ermöglicht es Benutzern, ein Konto zu registrieren, sich anzumelden und grundlegende Transaktionen durchzuführen, einschließlich Einzahlungen und Abhebungen. Es zeigt auch den Kontostand und die Transaktionshistorie an. Für einen einfachen Start wurde keine echte implemeniert sondern eine Klasse, die ähnliche Funktionen hat. Dies wird am Anfang des Programs immer wieder neu erstellt. Man muss also am Anfang neue Konten registieren.

# Anforderungen
1. Java Development Kit (JDK) 11 oder neuer
2. Ein kompatibles IDE, z.B. IntelliJ IDEA oder Eclipse

# Installation und Ausführung
einfach die Main-Klasse ausführen, um das Programm zu starten.

# Nutzung
Nach dem Start des Programms wird ein Anmeldefenster angezeigt. Nach der Registrierung kann man sich mit einer Benutzername und Passwort anmelden.

Im Hauptfenster des Programms kann man Geld einzahlen und abheben und den aktuellen Kontostand und die Transaktionshistorie einsehen.
